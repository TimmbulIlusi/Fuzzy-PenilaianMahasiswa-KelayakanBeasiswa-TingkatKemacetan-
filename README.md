# Aplikasi Logika Fuzzy

Aplikasi Streamlit yang berisi 3 studi kasus penerapan Logika Fuzzy.

## Studi Kasus

### 1. Penilaian Mahasiswa
Input:
- Nilai (0–100)

Output:
- Rendah
- Sedang
- Tinggi

### 2. Kelayakan Beasiswa
Input:
- IPK (0.00–4.00)

Output:
- Tidak Layak
- Dipertimbangkan
- Layak

### 3. Tingkat Kemacetan
Input:
- Jumlah Kendaraan (0–1000)

Output:
- Lancar
- Padat
- Macet

## Instalasi

```bash
pip install -r requirements.txt
```

## Menjalankan

```bash
streamlit run app.py
```

## Teknologi

- Python
- Streamlit
- NumPy
- Pandas
- Matplotlib
